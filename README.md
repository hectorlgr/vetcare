## VetCare - Microservicio de Gestión Veterinaria

Este es un microservicio backend desarrollado con Java Spring Boot para la gestión de atenciones en la clínica veterinaria VetCare.

## Funcionalidades
- CRUD de Atenciones: Registro, consulta, actualización y eliminación de registros.
- Consultas Especializadas: Filtrado por tipo de mascota, veterinario o fecha.
- Arquitectura: Patrón Controller-Service-Repository.
- Almacenamiento: Gestión de datos en listas.

## Tecnologías utilizadas
- Java 17+
- Spring Boot 3.x
- Lombok
- Maven

## Ejemplo de uso (API REST)
- 'GET /atenciones' - Obtener todas las atenciones.
- 'POST /atenciones' - Registrar una nueva consulta.
